# Boss: Mirefang Broodmother

## Act
- Act 1

## Role
- Arena denial controller

## Core Mechanics
- Toxic pool placement that shrinks safe space over time
- Periodic brood add waves that force target-priority decisions

## Encounter Summary
- Attrition-style fight where movement discipline matters more than burst.
- Main threat is overlap: pools plus add pressure plus boss swipe timing.

## Pattern Breakdown
- Venom Spit: targeted pool at current player location.
- Brood Call: summons small adds at interval checkpoints.
- Lunge Swipe: short forward burst to punish straight-line kiting.

## Telegraphs And Counters
- On poison charge-up animation, start moving before cast release to bait pool off-path.
- Clear nearest brood adds first to protect mobility lanes.
- If lunge starts, dodge sideways and reset to a clean area.

## Positioning Rules
- Rotate around arena edge in wide arcs, not tight circles.
- Never drop consecutive pools on the same escape lane.
- Keep boss and add wave on same screen side to avoid blind overlap.

## Kill Priority
- 1) Immediate melee brood on top of player.
- 2) Ranged brood threatening movement lane.
- 3) Boss damage only during pool-free windows.

## Common Failure Modes
- Tunnel damage on boss while pools stack behind player.
- Pulling fresh add waves into already narrowed zone.
- Using mobility too early and having no answer for lunge.
