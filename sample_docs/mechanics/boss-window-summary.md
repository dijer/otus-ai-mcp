# Path of Exile 2: Boss Window Summary

Boss fights are easiest when the player waits for a clear punish window.

## Rules

- Move first if the boss has not committed.
- Damage only when the boss is locked in recovery.
- Stop greed when the next phase begins.
- Burst is best when the boss cannot answer back.

## Summary

A clean kill comes from reading the fight, not forcing damage into unsafe space.
