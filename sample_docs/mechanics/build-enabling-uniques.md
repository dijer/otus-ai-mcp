# Path of Exile 2: Build Enabling Uniques Reference

## Part 1: Why Build Enablers Matter

Some unique items do not merely improve a build. They make a build possible.

Examples of what a unique can enable:
- A missing defense layer.
- A mana sustain loop.
- A damage conversion route.
- A movement or cooldown trick.
- A skill interaction the base class cannot provide alone.

## Part 2: Common Build Enabler Roles

### 2.1 Defense Enablers

These uniques solve survivability problems.

**Typical effects**:
- Large life bonus.
- Extra resistances.
- Armor or evasion boost.
- Immunity to a status ailment.

### 2.2 Offense Enablers

These uniques unlock damage scaling.

**Typical effects**:
- Added damage to attacks or spells.
- Conversion of damage type.
- Extra projectiles or area.
- Critical strike support.

### 2.3 Utility Enablers

These uniques make a build feel smoother.

**Typical effects**:
- Movement speed.
- Resource recovery.
- Cooldown reduction.
- Skill behavior changes.

## Part 3: Decision Rule

A unique is a build enabler if removing it breaks the build concept.

If the build still works without the item, it is a strong upgrade.
If the build collapses without the item, it is an enabler.

## Part 4: Practical Evaluation

Before chasing a unique, check:
- Does it solve a real problem?
- Is there a cheaper rare alternative?
- Does it scale with the rest of the build?
- Does it justify the opportunity cost?

If the answer to most of these is no, the item is probably a luxury, not a requirement.

## Conclusion

Build-enabling uniques should be treated as structural pieces of a build, not just stat sticks. The best unique items are the ones that let the build exist cleanly and consistently.
