# Path of Exile 2: Endgame Map Mods Mini Reference

## Part 1: Safe Mods

Safe mods are the ones that usually raise rewards more than they raise risk.

**Examples**:
- More monsters.
- More item quantity.
- More item rarity.
- More pack density.
- More basic currency.

These are usually acceptable if your build already clears comfortably.

## Part 2: Dangerous Mods

Dangerous mods raise risk enough that they can kill or slow the run.

**Examples**:
- Reduced recovery.
- Reduced resistances.
- Monster damage increased.
- Monster life increased.
- Extra projectiles.
- Ground hazards.

## Part 3: Build Sensitivity

Different builds care about different mods.

**Physical builds** hate armor reduction and reflect-like effects.
**Caster builds** hate mana pressure and cast speed penalties.
**Minion builds** hate minion damage reduction and heavy arena clutter.
**Ranged builds** hate projectile and movement constraints.

## Part 4: Roll or Run

Use a simple rule:
- If the map is safe and profitable, run it.
- If the map is risky but still clearable, run it only if the reward is high.
- If the map is dangerous and low value, reroll it.

## Conclusion

Map modifiers are not all equal. Learn which ones your build can ignore and which ones can turn a good map into a loss.
